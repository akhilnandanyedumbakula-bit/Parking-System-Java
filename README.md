# Parking-System-Java
This is a Smart Parking System built using Java. It allows users to reserve parking slots, view reservations, and calculate parking fees. The system also includes an admin module for checking available/booked slots and managing billing. User data is saved to a text file, and the system is compatible with Java 17+ and Java 21.
